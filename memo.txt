Authorization
    http 인증정보를 담기 위한 정보
    
    Authorization: Basic QWEJASKDJLKSDF==

Bearer
    JWT 기반의 인증 규칙
    이 토큰을 가지고 있는 사람은 인증된 사용자로 간주한다.

    Authorization: Bearer QWEJASKDJLKSDF==


Branch
    같은 프로젝트 안에서 서로 다른 개발 라인을 만들고 각자 개발 후 다시 합칠 수 있는 기능
    
    만드는 이유 : 
        1. 많은 테스트를 해도 메인이 망가질 염려가 없음
        2. 기능별/DB별 작업 내용을 분리
        3. 여러 명이 도잇에 다른 기능을 개발할 대 충돌을 줄임
        4. 각자의 개발 완료 후 main에 병합 가능

    브랜치 목록 보기
        git branch

    브랜치 생성 및 이동
    git switch -c mysql-support

    브랜치 이동
    git switch main

    기능별/브렌치명으로 적는게 과정
    git switch -c feature/mysql-support

    브랜치 삭제
    git branch -d mysql-support
